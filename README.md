## Misc Details
This branch is dedicated to miscellaneous files, experimental scripts, and assets that don't fit into the main SDK core.

###  What's in here?
* **Experimental CLI Tools**: Early versions of the project creation logic.
* **Asset Prototypes**: Icons, images, and UI mockups like `enginename.png`.
* **Alternative Mods**: Concepts for phone mods and non-Geode environments.

###  Note
Files in this branch may be unstable. For the stable **Synth SDK**, please switch to the `main` branch.
